# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

Be kind. Assume good faith. Disagree without belittling.

Not OK: harassment, personal attacks, demeaning comments (about anything — code, taste, identity, background), unwelcome sexual attention, doxxing, sustained disruption of discussions.

## Reporting

Email **[rinjanianalytics@gmail.com](mailto:rinjanianalytics@gmail.com)** — please don't file a public issue. We acknowledge within 2 business days, investigate confidentially, take action proportional to severity, and do not retaliate against good-faith reporters.

## Scope

All project spaces — issues, PRs, discussions, code comments, social channels, in-person events held under the project's name.
